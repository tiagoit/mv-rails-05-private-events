### Database structure
- users
  - name: STR, required, max:80
  - email: STR, required, index, unique, max:255

### Installation
- `bundle --without production`
- `yarn install --check-files`
- `rails db:migrate` 
- `rails db:seed`

### Star development server
- `rails server`

### Rails console
- `rails console`

### Integration tests
- `rails tests`
